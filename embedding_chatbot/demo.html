📋 Basic Integration

To embed the chatbot in your website, simply add this script tag:

<!-- Basic embedding with API key --> <script src="https://your-domain.com/embedding.js" apiKey="your-api-key"></script>

🎯 Custom Container

You can also specify a custom container for the chatbot:

<!-- HTML --> <div id="my-chatbot-container"></div> <!-- Script with custom container --> <script src="https://your-domain.com/embedding.js" apiKey="your-api-key" container="#my-chatbot-container"></script>

🔧 Manual Initialization

For more control, you can initialize the chatbot manually:

<script src="https://your-domain.com/embedding.js"></script> <script> // Manual initialization document.addEventListener('DOMContentLoaded', function() { DigitalProAssist.init('your-api-key', '#custom-container'); }); </script>

🌐 API Configuration

The chatbot communicates with the following endpoint:

POST https://n8n.anakanjeng.site/webhook-test/chatbot-digital-pro-assist Request Body: { "apiKey": "your-api-key", "message": "user message" } Expected Response: { "response": "bot response message" }
📝 Note: Replace "your-api-key" with the actual API key provided to your clients. Each client should have their own unique API key for tracking and security.
⚠️ Important: Make sure to host the embedding.js and chatbot-widgets.js files on your server and update the script src URLs accordingly.
